How To Play

The user interact with the display via command line, which is located at the bottom of the game windows.
The user have to type the command followed by (if any) the arguments and then press enter.

>>command <argument>

Here is the list of the possibles commands with a description

Name Arguments Description
help - Show all the possible commands in a precise moment of the game.
It can be always utilized.
newGame - Create a new game.
It can be utilized only in the menu.
